## üìñ Overview

**DonationTrack** is a cutting-edge decentralized crowdfunding platform that connects NGOs with donors worldwide. By leveraging blockchain technology, we ensure complete transparency, security, and accountability in the donation process. NGOs can create campaigns, receive funds, and provide proof of how donations are utilized, while donors can track their contributions in real-time and view exactly how their money is being used.

## üìã Prerequisites

Before you begin, ensure you have the following installed:
- Node.js (v14.0.0 or later)
- npm or yarn
- MetaMask wallet extension
- Ganache for local blockchain development
- Git

## üöÄ Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/DonationTrack-Blockchain-Crowdfunding.git
   cd DonationTrack-Blockchain-Crowdfunding
   ```

2. **Install dependencies**
   ```bash
   # Install frontend dependencies
   cd crowdfunding-ui
   npm install

   # Install smart contract dependencies
   cd ../smart-contracts
   npm install
   ```

3. **Configure environment**
   ```bash
   # In crowdfunding-ui directory
   cp .env.example .env
   ```
   Edit `.env` with your configuration:
   ```
   REACT_APP_FIREBASE_API_KEY=your_firebase_api_key
   REACT_APP_FIREBASE_AUTH_DOMAIN=your_firebase_auth_domain
   REACT_APP_FIREBASE_PROJECT_ID=your_firebase_project_id
   REACT_APP_FIREBASE_STORAGE_BUCKET=your_firebase_storage_bucket
   REACT_APP_FIREBASE_MESSAGING_SENDER_ID=your_messaging_sender_id
   REACT_APP_FIREBASE_APP_ID=your_app_id
   REACT_APP_CONTRACT_ADDRESS=your_deployed_contract_address
   ```

4. **Set up local blockchain**
   - Launch Ganache
   - Create a new workspace
   - Note the RPC Server URL (usually `http://127.0.0.1:7545`)
   - Import a Ganache account into MetaMask:
     - Copy a private key from Ganache
     - In MetaMask, click "Import Account" and paste the private key
     - Connect MetaMask to Ganache network (Network ID: 5777, RPC URL: http://127.0.0.1:7545)

5. **Deploy smart contracts**
   ```bash
   cd smart-contracts
   npx hardhat run scripts/deploy.js --network ganache
   ```
   ‚ö†Ô∏è Copy the deployed contract address and update it in your `.env` file.

6. **Launch the application**
   ```bash
   cd crowdfunding-ui
   npm start
   ```
   The application will be available at [http://localhost:3000](http://localhost:3000)

## üîß Usage

### üë®‚Äçüíº For NGOs

<details>
<summary>Click to expand NGO instructions</summary>

#### Registration & Profile Setup
1. Click "Sign Up" on the homepage
2. Complete the NGO registration form with valid credentials
3. Verify your email address through the sent link
4. Set up your NGO profile with:
   - Organization name
   - Description
   - Contact details
   - Profile image
   - Verification documents

#### Campaign Management
1. Log in to your NGO dashboard
2. Click "Create New Campaign" button
3. Fill in campaign details:
   - Campaign name
   - Description
   - Funding goal (in ETH)
   - Campaign duration
   - Category tags
4. Submit the campaign for approval

#### Fund Management
1. View all your campaigns in the dashboard
2. Track donations and campaign progress in real-time
3. For completed campaigns:
   - Submit proof of fund usage (documentation, receipts, photos)
   - Wait for verification
   - Withdraw funds to your wallet once verified

</details>

## üîí Security

DonationTrack implements multiple layers of security:

- **Smart Contract Audits**: All contracts audited for vulnerabilities
- **Multi-signature Wallets**: Key operations require multiple approvals
- **Proof Verification**: NGOs must provide evidence of fund usage
- **Data Encryption**: Sensitive user data is encrypted
- **Regular Security Updates**: Continuous monitoring and patching
